# Brew Haven ☕

Brew Haven is a coffee shop e-commerce website that allows customers to browse products, create accounts, place orders, and manage their profiles.

## Features

### Customer Features
- Account Registration
- Login System
- Profile Management
- Loyalty Rewards
- Shopping Cart
- Checkout Process
- Receipt Generation

### Shopping Features
- Hot Drinks Menu
- Cold Drinks Menu
- Fresh Pastries Menu
- Product Search Bar
- Cart Counter Badge

### Information Pages
- Home Page
- About Us
- Contact Us

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
